How to build for bacon

Setting up the source

Step 1: Create a new directory.

mkdir -p ~/aosparadox/
cd ~/aosparadox/

Step 2: Initialize the CAF repo.

repo init -u git://codeaurora.org/platform/manifest.git -b release -m default_LA.BF.1.1.3-01010-8x74.0.xml

Step 3: Add our github repositories.

mkdir -p .repo/local_manifests/
wget -O .repo/local_manifests/bacon_par.xml https://raw.githubusercontent.com/AOSParadox/local_aosparadox/bacon-par-6.0/bacon_par.xml

Step 4: Sync

repo sync -c --no-tags -j[Amount of CPU threads]


Building the ROM

Step 5: Build

. build/envsetup.sh
lunch full_bacon-user
make -j[Amount of CPU threads] otapackage

You're done! If no errors occured, the build is available at out/target/product/bacon/
